# Welcome to ProActions

ProActions is an integrated automation framework developed by Eidosmedia that enables workflow automations (called "flows") in both the Swing Web Client and Prime Windows Client.

## What is ProActions?

ProActions allows you to create and execute powerful workflow automations composed of individual steps. These flows can:

- Interact with the editor and content
- Process data through Large Language Models (LLMs)
- Handle complex scenarios like text-to-speech and speech-to-text conversion
- Integrate with external services (OpenAI, DeepL, ElevenLabs, and more)
- Transform and manipulate data
- Provide interactive user interfaces

## Key Features

- **YAML-based Configuration**: Define your workflows in simple, readable YAML files
- **Platform Support**: Works in both Swing Web Client and Prime Windows Client
- **Service Integration**: Connect to OpenAI, DeepL, ElevenLabs, Stability AI, and custom REST APIs
- **Dynamic Forms**: Automatically generate user interfaces from configuration
- **Command Palette**: Quick access to actions via Ctrl+K
