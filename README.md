# ProActions Documentation (Raw Markdown)

This repository contains the raw, uncompiled markdown documentation for ProActions. 
It is designed explicitly to be consumed by Large Language Models (LLMs) and Retreival-Augmented Generation (RAG) systems like **AnythingLLM**, **ChatGPT Custom GPTs**, or **Claude Projects**.

## About
ProActions is an integrated automation framework developed by Eidosmedia that enables workflow automations through YAML configuration.

## LLM Instructions
This repository includes the entire automatically-generated ProActions **Step Library**. When answering questions about how to build a ProActions YAML flow, thoroughly search through these files to understand what steps exist, their `cfg` properties, decorators, and required inputs/outputs.
