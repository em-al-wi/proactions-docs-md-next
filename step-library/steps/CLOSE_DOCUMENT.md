# CLOSE_DOCUMENT

Closes the current document without saving.

## At a glance
- **Category** Editor
- **Version:** 1.2.0
- **Applications:** all
- **Scope:** all

## Examples

### Close the current document
```yaml
- step: CLOSE_DOCUMENT
```
