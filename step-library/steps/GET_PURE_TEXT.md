# GET_PURE_TEXT

Retrieves the full pure text content from the current document and sets it as output.

## At a glance
- **Category** Editor
- **Version:** 1.2.0
- **Applications:** all
- **Scope:** all

## Outputs
| Type | Description | Optional |
|---|---|:---:|
| `text` | The pure text content of the document or report. | false |

## Examples

### Get full document pure text
```yaml
 - step: GET_PURE_TEXT
```
